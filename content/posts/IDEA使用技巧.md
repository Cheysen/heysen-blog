# 1.快捷键

| 快捷键                 | 说明                               |
| ---------------------- | ---------------------------------- |
| Ctrl + e               | 查看最近的文件                     |
| Ctrl + b               | 转到声明                           |
| F2                     | 光标跳转到下一个建议或警告         |
| Ctrl + Alt + b         | 转到实现                           |
| Alt + F7               | 查找所有的使用                     |
| 双击Ctrl               | 运行任何内容                       |
| Ctrl + w               | 延伸选择区域                       |
| Ctrl + shift + w       | 缩小选择区域                       |
| Ctrl + shift + enter   | 完成当前语句并将光标调到正确的位置 |
| Ctrl + shift + alt + t | 重构                               |
| Ctrl + shift + a       | 查找操作                           |
| 双击shift              | 全局搜索                           |
